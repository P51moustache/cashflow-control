# Cashflow Control

A personal cashflow and budgeting app for iOS and Android. Track income and expenses, see where your money goes, and stay ahead of your spending. Built with React Native (Expo) and Supabase.

## Features
- Quick transaction entry with categories and document import
- Spending and cashflow charts and trends
- Secure accounts: email and Sign in with Apple, synced through Supabase
- Guided onboarding and a subscription paywall (RevenueCat)
- Polished native experience: haptics, gestures, and offline-friendly local storage

## Tech
React Native · Expo (expo-router) · TypeScript · Supabase (Postgres + Auth) · NativeWind / Tailwind · RevenueCat · PostHog · Sentry

## Run it
```bash
npm install
npx expo start
```
Supabase and RevenueCat keys go in a local `.env`.
